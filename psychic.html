Instruction

This app randomly picks a letter, and the user has to guess which letter the app choses

  • The player picks a random letter
  • The computer makes a guess
  • If the player's choice matches the computer choice, the player wins
  • The user is allowed to make ten gueses
  • When the player wins, the wins counter increases and prompt the user to continue or not
  • The game restarts without a page refresh if the player types yes
  • When the player loses, the Losses counter increases and prompt the user to continue or not
  • The game restarts without a page refresh if the player types yes
